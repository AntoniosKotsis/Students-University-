# Students-University-
Management of Students from any University

**Student Enrollment Management Program at a University**

Supports registration, viewing, searching, deletion, and updating of student information.

1. **Registration**

   * Add a new student record (insert **NOT in the middle** of the list).
   * **Student information fields**: full name, age, gender, faculty, department, student ID, year of enrollment, and semester.

2. **Viewing (Display)**

   * Display all student records.

3. **Search**

   * Search by **student ID** using a **vector**.
   * If the student ID exists, display the corresponding student’s information.

4. **Deletion**

   * Delete a student record based on **student ID** (using a **list**).

5. **Update Student Information**

   * Based on **student ID**, allow updating **age, year of enrollment, and semester**.

**Data Storage:**

* For registration, deletion, and updating (points 1, 4, 5), all student data will be stored in a **text file (.txt) or CSV file**.
